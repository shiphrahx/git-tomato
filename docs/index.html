How it works

Focus. Commit. Level up.

git-tomato turns your actual coding output into a progression system. Every reward traces back to a git log entry.

01
🍅
Start a session
Pick a task, start a Pomodoro.
02
⌨️
Write code
Work normally. git-tomato scans your watched repos for commits the moment your session ends.
03
📊
See the truth
Your git activity appears in the feed. XP is awarded per commit, per session, per streak day.
04
🏅
Earn rewards
Badges, levels, streaks, and daily quests all gated behind real output.

Day & Night

Two themes.
One vibe.

Switch between day and twilight modes. The pixel-art background gives it a cosy, gamey vibe that makes deep work feel a bit less painful.

Day mode Night mode Pixel art backgrounds

Progress

XP that means
something.

Seven developer levels from Seedling to Legend. A 12-week focus heatmap. A live day streak that only counts when you have commits to show for it.

7 levels Day streaks Focus heatmap Commit-gated XP

Daily Quests

Challenges that
scale with you.

Three quests generated fresh each day, calibrated to your own 7-day baseline. Unfinished quests expire at midnight. Here we don't carry over yesterday's debt.

Adaptive targets 3 quests / day Midnight reset 14 quest types

Achievements

25 badges.

One-time unlocks across five categories: Output, Consistency, Time, Style, and Mastery. Badges like Ghost Mode (never paused the timer), The Refactorer (only deletions), and Comeback Kid (returned after 7 days away) tell a story about how you actually work.

25 badges 5 categories Pixel art icons
